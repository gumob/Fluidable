FluidNavigationActionDelegate

public protocol FluidNavigationActionDelegate : FluidDelegate

The methods in this protocol let you control views while running the navigation.

  • navigationPresentAnimationDidProgress(from:to:with:on:navigationStyle:duration:easing:state:progress:)

    The CGFloat value indicating the current progress of the navigation.
