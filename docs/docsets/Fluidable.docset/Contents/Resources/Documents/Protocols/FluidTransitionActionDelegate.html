FluidTransitionActionDelegate

public protocol FluidTransitionActionDelegate : FluidDelegate

The methods in this protocol let you control views while running the transition.

  • transitionPresentAnimationDidProgress(from:to:with:on:transitionStyle:duration:easing:state:progress:)

    The function that is invoked when an animated presentation transition updates.

    func transitionPresentAnimationDidProgress(from source: FluidSourceViewController, to destination: FluidDestinationViewController, with navigation: FluidNavigationController?, on container: UIView?, transitionStyle: FluidTransitionStyle, duration: TimeInterval, easing: FluidAnimatorEasing, state: FluidProgressState, progress: CGFloat)

    Parameters

    source

    The source view controller that conforms to FluidSourceViewController.
    destination

    The destination view controller that conforms to FluidDestinationViewController.
    navigation

    The destination navigation view controller that conforms to FluidDestinationNavigationController.
    container

    The UIView object in which the animated transition should take place.
    transitionStyle

    The FluidTransitionStyle value.
    duration

    The TimeInterval value.
    easing

    The FluidAnimatorEasing value.
    state

    The FluidAnimatedTransitionState value indicating the current state of the transition.
    progress

    The CGFloat value indicating the current progress of the transition.
